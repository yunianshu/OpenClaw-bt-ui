⚠️ 常见误区

不要给 Agent 加载过多工具。从简单开始，根据实际需求逐步添加工具。

建议：每个 Agent 最多 5-7 个工具以获得最佳性能。

常见陷阱

⚠️ 上下文溢出

超出上下文窗口会导致响应质量下降。对长对话实施摘要压缩。

⚠️ Token 预算泄漏

没有监控会导致 Token 使用超出预算。设置警报和自动限流。
