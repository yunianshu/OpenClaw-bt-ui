常见陷阱

⚠ Token 成本失控

上下文过长会导致费用激增。建议：
• 开启上下文自动压缩
• 设置单次对话 Token 上限
• 使用缓存减少重复计算

更多注意事项

⚠ 幻觉问题

AI 可能生成看似合理但错误的内容。
关键场景务必开启结果审核。

⚠ 隐私泄露

敏感数据不要直接发送给大模型。
使用本地向量库替代云端检索。
